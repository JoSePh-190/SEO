Who I Am

Engineer by Training. Developer by Practice.

My name is Gbarabe Tordi Joseph. I am a final-year Petroleum Engineering student at the University of Port Harcourt, one of Nigeria's foremost centres for engineering education. Throughout my undergraduate journey, I have developed a rigorous grounding in the science of subsurface fluid behaviour, reservoir characterisation, well performance, and enhanced oil recovery — disciplines that demand both analytical precision and creative problem-solving.

Alongside my engineering studies, I work professionally as a software developer. I build web applications and computational tools in Python and Django, translating complex engineering workflows into software that engineers and students can use directly in the browser. My work sits at a rare intersection: I understand the physics deeply enough to model it correctly, and I understand software well enough to ship it cleanly.

Featured Work

Tools I've Built

Every project I build solves a real problem I encountered in my engineering studies or in working with other students. The goal is always the same: replace a laborious spreadsheet process with something faster, more transparent, and shareable.

MBE Calculator Solver

Reservoir Engineering

A web-based Material Balance Equation solver that accepts reservoir production history and fluid PVT data to estimate original fluids in place and identify the dominant drive mechanism. Eliminates the need for complex manual iteration and provides instant graphical diagnostics — Havlena-Odeh and F vs Eo plots rendered in the browser.

Python Django NumPy Matplotlib

EOR Calculator Solver

Enhanced Oil Recovery

An Enhanced Oil Recovery screening and calculation tool that evaluates candidate reservoirs against established technical criteria for water flooding, polymer flooding, CO₂ miscible injection, and thermal methods. Outputs a ranked suitability report and incremental recovery estimates, helping engineers make fast, evidence-based decisions during field development planning.

Python Django SciPy REST API

Engineering Web Applications

Full-Stack

A suite of purpose-built Django web applications for petroleum and reservoir engineering education. Each application provides an interactive interface for a specific calculation workflow — from Darcy's Law permeability estimation to nodal analysis and wellbore pressure loss — making engineering computation accessible without any specialist software installation.

How I Work

My Approach

⚙

Engineering First

Every tool I build starts from the engineering problem, not the technology. I make sure the underlying physics or mathematics is correct before a single line of framework code is written.

📐

Clean Architecture

I write code that can be read, maintained, and extended. Django's MTV pattern keeps my projects organised, and I test calculation logic independently from the presentation layer.

🎯

User-Centred Design

My users are engineers and students, not necessarily technical users. I design interfaces that guide the user through a workflow without requiring them to understand the code behind it.

📈

Continuous Learning

Both petroleum engineering and software development evolve quickly. I stay current across both fields — following advances in reservoir simulation on one side and modern Python tooling on the other.
